SD安装文档

Stable Diffusion 安装小说明

学姐专用SD安装文档 v0.1

1. 🗺 安装环境准备

1.1 🍺Homebrew安装

Homebrew是一款Mac OS平台下的软件包管理工具，拥有安装、卸载、更新、查看、搜索等很多实用的功能。在本次安装之旅中扮演了安装python以及一些乱七巴糟的库的角色（什么巴库？）。好！现在出发8心老师！

①. 把它收入囊中吧！！打开📺Terminal输入下边命令。期间按照提示输入密码。当出现 ==> Installation successful!时，恭喜安装成功。

②. 来看看超简单的基本命令吧！！在📺Terminal里输入哦！

  • 软件安装：brew install 软件名

    举个栗子：brew install wget

  • 软件卸载：brew uninstall 软件名

    举个栗子：brew uninstall wget

  • 都装了啥啊啊啊啊啊：brew list

  • 遇到困难睡大觉：brew help

1.2 🥁学姐逃不掉的Python3.10一家子安装

主要是Python以及python下实用的工具库。

直接开始咏唱魔法！继续在📺Terminal里告诉你的🍺brew小跟班该安装些啥吧！！！

就这么简单，完事儿了已经。

1.3 ✨喜提Stable Diffusion的Web的UI

Stable Diffusion只是一个AI模型，大伙儿们一起建了个界面，是基于Web的UI，所以叫做webui！

①. 📺Terminal里，先去你的办公桌，新建一个叫aigc的文件夹，这就是Stable Diffusion待会儿下载的地方。

②. 用一个叫做git的工具从基佬霍卜上把这个项目的Repository给下载下来，拎包入住。

现在进房间看看吧，打开它！！
